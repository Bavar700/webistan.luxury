# Академия - WordPress Theme

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![WordPress](https://img.shields.io/badge/WordPress-5.8+-green.svg)
![PHP](https://img.shields.io/badge/PHP-7.4+-purple.svg)

Профессиональная тема WordPress для образовательных проектов и академических сайтов, специализирующихся на изучении древних языков и культур.

## 🎨 Особенности дизайна

### Цветовая схема
Вдохновлена древними манускриптами и академической эстетикой:

- **Основной цвет**: `#1a2332` - Глубокий синий (чернила древних манускриптов)
- **Акцентный цвет**: `#c9a961` - Золотой (позолота древних книг)
- **Дополнительный**: `#8b4513` - Коричневый (старая кожа переплетов)
- **Фон**: `#f8f6f1` - Пергаментный оттенок

### Типографика
- **Заголовки**: Georgia, Palatino Linotype, Book Antiqua (классические serif шрифты)
- **Основной текст**: System fonts для оптимальной читабельности

## 📦 Структура темы

```
Academy_Webistan/

├── footer.php                # Подвал сайта

├── front-page.php            # Шаблон главной страницы

├── sidebar.php               # Боковая панель

├── archive-archive.php       # Архив медиа-материалов

```

## ✨ Функциональность

### Custom Post Types (Произвольные типы записей)

#### 1. Resources (Библиотека)
- **Назначение**: PDF-документы, научные статьи, исследования
- **URL**: `/resources/`
- **Иконка**: 📚 (dashicons-book-alt)
- **Таксономии**: 
  - `resource_category` - Категории ресурсов
  - `resource_tag` - Метки ресурсов

#### 2. Archive (Медиа-архив)
- **Назначение**: Фото, аудио и видео материалы
- **URL**: `/archive/`
- **Иконка**: 🎬 (dashicons-format-gallery)
- **Таксономии**:
  - `archive_category` - Типы материалов (Фото, Аудио, Видео)
  - `archive_tag` - Метки материалов

### Области виджетов

1. **sidebar-1** - Основная боковая панель
2. **footer-1** - Первая колонка подвала
3. **footer-2** - Вторая колонка подвала
4. **footer-3** - Третья колонка подвала

### Меню навигации

1. **primary** - Главное меню (в шапке)
2. **footer** - Меню в подвале

## 🏠 Главная страница (Front Page)

Структура главной страницы включает:

1. **Hero Section** - Приветственный блок с градиентным фоном
2. **About Section** - Описание проекта
3. **Features Grid** - Три ключевых направления (CSS Grid)
4. **Recent Resources** - Последние материалы библиотеки
5. **Archive Showcase** - Избранные медиа-материалы
6. **CTA Section** - Призыв к действию

## 🌐 Поддержка UTF-8

Тема полностью поддерживает UTF-8 кодировку для корректного отображения:
- Древних языков (клинопись, иероглифы, руны)
- Кириллицы
- Латиницы с диакритическими знаками
- Любых Unicode символов

## 📱 Адаптивность

Тема полностью адаптивна и оптимизирована для:
- 📱 Мобильных устройств (< 768px)
- 📱 Планшетов (768px - 992px)
- 💻 Десктопов (> 992px)

## 🚀 Установка

1. Скопируйте папку темы в `/wp-content/themes/`
2. Войдите в админ-панель WordPress
3. Перейдите в **Внешний вид → Темы**
4. Активируйте тему "Академия"

## ⚙️ Настройка

### Рекомендуемые плагины

Для полноценной работы рекомендуется установить:
- **Advanced Custom Fields** - для дополнительных полей
- **Yoast SEO** - для SEO-оптимизации
- **Contact Form 7** - для форм обратной связи

### Создание меню

1. Перейдите в **Внешний вид → Меню**
2. Создайте новое меню
3. Назначьте его в позицию "Главное меню" или "Меню в подвале"

### Настройка виджетов

1. Перейдите в **Внешний вид → Виджеты**
2. Добавьте виджеты в доступные области:
   - Боковая панель
   - Подвал - Колонка 1, 2, 3

### Настройка главной страницы

1. Создайте новую страницу
2. Перейдите в **Настройки → Чтение**
3. Выберите "Статическая страница"
4. Укажите созданную страницу как главную

## 🎨 Кастомизация

### Изменение цветов

Отредактируйте CSS-переменные в `style.css`:

```css
:root {
    --primary-color: #1a2332;
    --secondary-color: #c9a961;
    --accent-color: #8b4513;
    /* ... */
}
```

### Добавление пользовательских стилей

Используйте **Внешний вид → Настроить → Дополнительные CSS** или создайте дочернюю тему.
